# AI-Powered Women's Health App

This repository contains a complete structure for an AI-powered women's health application that includes both frontend and backend components.

## Project Structure
- Frontend: Next.js with TypeScript and Tailwind CSS
- Backend: Node.js with Express and TypeScript
- AI/ML: Python backend with Flask for ML models
- Database: MongoDB

## Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/rjrandria-lang/ai-forwomen.git
   cd ai-forwomen
   ```
2. Run the application using Docker:
   ```bash
   docker-compose up
   ```
